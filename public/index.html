📢 ตั้งค่าแจ้งเตือน Telegram

แจ้งเตือนเมื่อทำกิจกรรมครบเวลาที่กำหนด และเมื่อเกินเวลาที่ admin ตั้งเพิ่ม

🤖 Bot Token (จาก @BotFather)
💬 Chat ID (ของกลุ่ม Telegram)
กลุ่มต้องเพิ่ม Bot เป็นสมาชิกก่อน (Group → Add members → ใส่ @bot_name)
⏰ แจ้งเตือนระดับ 2: เกินกำหนดกี่นาที
นาที (บวกจาก limit ของกิจกรรม)
🔔 ระดับ 1 แจ้งทันทีเมื่อครบ limit (เช่น สูบบุหรี่ครบ 20 นาที)
🚨 ระดับ 2 แจ้งอีกครั้งเมื่อเกิน limit + ค่านี้ (เช่น 20+3 = 23 นาที)
🔔 สถานะ
📋 วิธีตั้งค่าครั้งแรก:
1. ใน Telegram → ค้นหา @BotFather → /newbot → ตั้งชื่อ → ได้ Token
2. สร้างกลุ่ม → เพิ่ม Bot เป็นสมาชิก
3. ส่งข้อความใดก็ได้ในกลุ่ม → เปิด https://api.telegram.org/bot[TOKEN]/getUpdates → หา chat.id (เลขลบของกลุ่ม)
4. ใน Apps Script → Run ฟังก์ชัน setupOvertimeTrigger ครั้งเดียว (สร้าง trigger สแกนเกินเวลา)
